# F01 ほしい物リスト - テストケース

## ユニットテストケース

### ImageProcessor（`ImageProcessorTests.swift`）

| テストメソッド | 説明 |
|---|---|
| `testCompressReturnsDataForValidInput` | 有効なJPEGデータを渡すと、圧縮済みDataが返る |
| `testCompressReturnsNilForInvalidData` | 画像として解釈できない無効なDataを渡すと nil が返る |
| `testCompressResizesImageExceedingMaxDimension` | 長辺が maxDimension を超える画像は、指定サイズ内に縮小される |
| `testCompressKeepsSizeWhenBelowMaxDimension` | 長辺が maxDimension 以下の画像は、元のサイズのまま返る |
| `testResizedPreservesAspectRatio` | リサイズ後もアスペクト比（幅:高さ）が維持される |
| `testResizedPortraitImageFitsWithinMaxDimension` | 縦長画像の長辺が maxDimension 以内に収まる |
| `testResizedReturnsOriginalWhenBelowMaxDimension` | maxDimension 以下の小さい画像はリサイズせずそのまま返る |

## 手動確認項目

### 画像選択・表示

| # | 操作 | 期待結果 |
|---|------|----------|
| 1 | 登録・編集画面の「画像」セクションをタップ | フォトライブラリが開く |
| 2 | 画像を選択して保存 | 詳細画面の上部に画像が表示される |
| 3 | 詳細画面の画像表示サイズ | 画面高さの約25%で表示される |
| 4 | 編集画面で既存の画像が表示される | 登録済み画像がプレビュー表示される |
| 5 | 「画像を削除」ボタンタップ後に保存 | 詳細画面で画像が非表示になる |
| 6 | 画像なしで保存 | 詳細画面に画像セクションが表示されない |
