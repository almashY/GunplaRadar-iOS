## S01 ほしい物リスト画面

**遷移元**: タブバー

### レイアウト

```
┌─────────────────────────┐
│ ほしい物リスト    ＋登録 │  ← NavigationBar
├─────────────────────────┤
│ 🔍 検索バー             │
├─────────────────────────┤
│ ソートキー（メニュー）   │
├─────────────────────────┤
│ ┌─────────────────────┐ │
│ │ ▌ 商品名       [画像]│ │
│ │   グレード           │ │  ← GunplaItemRow
│ │   値段  発売日       │ │
│ └─────────────────────┘ │
│  ...繰り返し...          │
└─────────────────────────┘
```

| 要素 | 仕様 |
|------|------|
| 登録ボタン | NavigationBar 右上。タップで S02b へ遷移 |
| 検索バー | リアルタイム絞り込み |
| ソートキー | メニュー形式で選択 |
| アイテム行 | 左端に優先度カラー帯、右端にサムネイル画像（56×56pt）。タップで S02a（詳細）へ遷移、スワイプで削除 |
| 優先度カラー帯 | 最高=赤・高=オレンジ・中=緑・低=青 |
| 空状態 | アイテムが 0 件の場合「アイテムがありません」を表示 |

## S02b ガンプラ登録・編集画面

**遷移元**: S01 登録ボタン（新規） / S02a 編集ボタン（編集）

### レイアウト

```
┌─────────────────────────┐
│ ← 戻る    登録/編集  保存│
├─────────────────────────┤
│ 商品名 *         [入力] │
│ グレード *        [選択] │
│ 優先度 *          [選択] │
│ 値段             [入力] │
│ URL              [入力] │
├─────────────────────────┤
│ 画像                     │
│     [ 画像プレビュー ]   │  ← 画面幅80%・中央揃え・選択/削除可
├─────────────────────────┤
│ 発売日           [日付選択] │
│ 次回再販日       [日付選択] │
│ 購入日           [日付選択] │
├─────────────────────────┤
│ 優先度  低 中 高 最高   │  ← セグメント（右が最高）
└─────────────────────────┘
```

| 要素 | 仕様 |
|------|------|
| 必須項目（*） | 未入力時に保存ボタンを非活性にする |
| グレード選択 | RE / PG / MG / RG / HG / EG / その他 の固定リストからドラムロール（Picker）で選択。「その他」選択時はテキスト入力欄を表示 |
| 画像選択 | PhotosPicker でフォトライブラリから選択。選択済みの場合はプレビュー（画面幅80%・中央揃え）を表示。「画像を削除」ボタンで削除可能 |
| 日付選択 | タップで DatePicker（カレンダー形式）をシートで表示 |
| 優先度 | セグメント形式。左から 低・中・高・最高 の順（右が最高） |
| 優先度 | セグメント形式。左から 低・中・高・最高 の順（右が最高）。選択した優先度の色（最高=赤・高=オレンジ・中=緑・低=青）が各画面の左帯・カレンダードットに反映 |
| 購入場所 | タップで登録済み店舗一覧シートを表示、お気に入り店舗を優先表示 |

## S03 再販カレンダー画面

**遷移元**: タブバー

### レイアウト

```
┌─────────────────────────┐
│        カレンダー        │  ← NavigationBar（固定）
├─────────────────────────┤
│  ＜ 2025年 4月 ＞       │
│ 日 月 火 水 木 金 土    │
│  1  2  3  4  5  6  7   │
│ [①] . [②] .  ③  .  .  │  ①=再販（赤字+赤リング） ②=再販ドット ③=巡回（太字）
│  ...           今月の購入予定金額 ¥○○○○ │  ← 右寄せ
├─────────────────────────┤
│ ○年○月○日              │
│ ┌───────────────────┐   │
│ │ ▌ 商品名  ● [画像] │🕐│ │  ← 行タップで詳細遷移 / 🕐=品出し記録ボタン
│ └───────────────────┘   │
│ 🚶 巡回予定             │
└─────────────────────────┘
```

| 要素 | 仕様 |
|------|------|
| ● ドット | 再販日が登録されているアイテムの優先度カラーでドット表示（最高=赤・高=オレンジ・中=緑・低=青）。複数アイテムがある日は複数ドット（最大3つ） |
| 再販日の強調 | 日付数字を赤字・赤いリングで強調表示 |
| 巡回予定の表示 | 日付数字を太字で表示 |
| 月合計金額 | カレンダーグリッド右下に表示。設定画面で選択した優先度かつ価格登録済みのアイテムのみ集計。対象アイテムが0件の場合は非表示 |
| 日付タップ | 画面下部にアイテム一覧を展開表示 |
| アイテム行 | 優先度カラー帯・商品名・グレード・サムネイル。行タップで詳細画面（S02a）へ遷移 |
| 品出し記録ボタン | アイテム行右端にオレンジのボタンを表示。タップでアイテム選択済みの品出し差分登録画面（S05）へ遷移 |

## S06 店舗分析画面

**遷移元**: タブバー

### レイアウト

```
┌─────────────────────────┐
│ 店舗分析                 │
├─────────────────────────┤
│ 🔍 場所・住所・店名を検索│  ← 入力中に候補リスト表示
│ ┌───────────────────┐   │
│ │ 候補1（住所）      │   │
│ │ 候補2（店名）      │   │  ← MKLocalSearchCompleter
│ └───────────────────┘   │
│                         │
│     [MapKit 地図]       │  ← ピン = 登録済み店舗
│                📍  📋   │  ← 現在地ボタン・店舗一覧ボタン
└─────────────────────────┘
```

| 要素 | 仕様 |
|------|------|
| 検索バー | `MKLocalSearchCompleter` によるリアルタイム入力補完。地名・住所・店舗名・POI名すべて対応 |
| 候補リスト | 入力中に最大10件程度の補完候補を表示。候補タップで S06a（場所詳細）を表示 |

| 現在地ボタン | 右下に固定表示。タップで現在地にカメラ移動。現在地は青いドットで常時表示 |
| 登録済みピンタップ | S07（登録済み店舗詳細）を表示 |
| マップ上のPOIタップ | S06a（場所詳細）を表示 |
| 店舗一覧ボタン | 右下に固定表示。タップで S08 へ遷移 |

## S13 品出し時刻記録画面

**遷移元**: S07「編集」タップ / S08 行タップ / S11 店舗行タップ

```
┌─────────────────────────┐
│ ← 店舗名          編集  │  ← NavigationBar
├─────────────────────────┤
│ 品出し時刻（N / 5件）    │  ← セクションヘッダ
│  🕐 08:30          🗑   │  ← editモード中のみ🗑表示
│  🕐 10:15          🗑   │
│  🕐 13:00          🗑   │
│  ── 時刻を追加 ──        │  ← editモード時のみ表示（上限未満）
├─────────────────────────┤
│ 品出し差分               │
│  平均差分     +2.5時間   │
│  🗑 差分データをリセット  │
└─────────────────────────┘
```

| 要素 | 仕様 |
|------|------|
| 一覧 | 登録済み時刻を HH:mm 形式で表示（最大5件） |
| 編集ボタン | NavigationBar 右上。タップで editモード切替（「編集」⇔「完了」） |
| 時刻を追加 | editモード中・5件未満の場合に表示。タップでドラムロール時刻ピッカーシートを表示 |
| 削除（スワイプ） | 左スワイプで「削除」ボタンを表示。即削除 |
| 削除（ゴミ箱ボタン） | editモード中のみ各行右端にゴミ箱アイコンを表示。タップで確認ダイアログ → OKで削除 |
| 時刻ピッカー | `.datePickerStyle(.wheel)` によるドラムロール形式（時・分のみ） |
| 平均差分 | 差分記録がない場合は「記録なし」、ある場合は `+X.X時間` 形式で表示 |
| 差分データをリセット | タップで確認ダイアログ → OKでこの店舗の全差分データを削除し平均を0にリセット |

## S14 対象優先度選択画面

**遷移元**: S12「対象優先度」セルタップ

| 要素 | 仕様 |
|------|------|
| 一覧 | 低・中・高・最高 の4行をリスト表示 |
| 複数選択 | 各行をタップするたびにチェックマークをON/OFF。選択は UserDefaults にビットマスク（Int）で即時永続化 |
| デフォルト | 全優先度が選択済み（ビットマスク = 15） |
