# F01 ほしい物リスト

ガンプラ情報の登録・管理・ソート・検索を行う機能。

---

## 新規登録

登録ボタン押下により、以下の情報を入力可能とする。

| 項目 | 必須 | 備考 |
|------|------|------|
| 画像 | - | フォトライブラリから選択（PhotosPicker）。選択後は自動でリサイズ・圧縮して保存。削除も可能 |
| URL | - | Amazon・プレバン等 |
| 商品名 | ○ | |
| グレード | ○ | RE / PG / MG / RG / HG / EG / その他 の固定リストからドラムロールで選択。「その他」選択時はテキスト入力 |
| 値段 | - | |
| 次回再販日 | - | 再販カレンダーと連携。入力欄タップでカレンダーUIを表示して選択 |
| 発売日 | - | 入力欄タップでカレンダーUIを表示して選択 |
| 購入日 | - | |
| 購入場所 | - | 店舗分析機能と連携 |
| 優先度 | ○ | 低・中・高・最高の4段階。左が低・右が最高 |
| タグの色 | - | |

日付入力欄（発売日・再販日）はテキスト直接入力ではなく、タップ操作でDatePickerダイアログ（カレンダーUI）を表示し、日付を選択して入力する。

---

## 過去情報からの作成

「過去の情報から作成」ボタン押下で登録済み情報を検索・選択可能。
選択すると以下の基本情報が自動入力される。

- 画像
- URL
- 商品名
- グレード
- 値段
- 発売日
- 購入日

---

## ソート機能

複数ソートに対応し、左から順に優先度が高い。

| ソートキー | 備考 |
|-----------|------|
| 優先度順 | |
| 再販日順 | 現在日時に最も近い順。再販日が過去のものは非表示 |
| 発売日順 | |
| グレード順 | |
| 値段順 | |

---

## 検索機能

商品名・グレード・タグ等をキーワードで絞り込み検索できる。

---

## アイテム詳細表示

リストのアイテムをタップすると詳細画面へ遷移し、以下の情報を確認できる。

| 項目 | 備考 |
|------|------|
| 商品名・グレード | タグカラーとともに表示 |
| 価格・優先度 | |
| 発売日・再販日・購入日 | |
| URL | タップでブラウザ（Safari等）に遷移 |
| 画像 | 詳細情報セクションの下に表示。画面幅の60%・中央揃え・見切れなし |

詳細画面右上に「編集」ボタンを表示し、タップで編集画面へ遷移する。
編集画面右上の「保存」ボタンで変更を保存する。
