# 要件定義書 - Gunpla Radar

## アプリ名
**Gunpla Radar**

---

## 1. 概要

### 1.1 目的
ガンプラ再販品の品出し日は店舗によって差異があり、再販予測日と実際の品出し日のずれを蓄積・分析することで、より効率的にガンプラを入手できる環境を提供する。また、再販日の記録や模型屋の管理を地図・カレンダーアプリとは独立して行えるツールとして機能させる。

### 1.2 対象ユーザー
ガンプラの再販情報を管理・追跡したいコレクター・購入者

### 1.3 開発環境
- プラットフォーム：iOS
- フレームワーク：SwiftUI

## 3. 非機能要件

| 項目 | 内容 |
|------|------|
| データ保存 | ローカル永続化（SwiftData） |
| 地図 | MapKit を使用 |
| 通知 | UserNotifications フレームワーク使用 |
| 対応OS | iOS 17.0 以上（SwiftUI・SwiftData活用のため） |
| オフライン動作 | 基本機能はオフラインで動作すること |
