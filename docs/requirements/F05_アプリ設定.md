# F05 アプリ設定

通知設定・蓄積データ管理を行う機能。

---

## 設定項目

| 設定項目 | 内容 |
|----------|------|
| 通知設定 | 巡回予定通知のON/OFF切り替え |
| 再販購入予定金額 | カレンダーの月合計金額に含める優先度を複数選択。低・中・高・最高それぞれを個別にON/OFF可能。設定は UserDefaults（ビットマスク）に永続化 |
| 蓄積データ管理 | 店舗分析の品出し差分データを削除可能。一括削除または編集モードで複数選択削除に対応。 |
