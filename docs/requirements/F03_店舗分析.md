# F03 店舗分析

店舗登録・お気に入り管理・品出しずれ分析を行う機能。

---

## 画面構成

- 上部：検索バー（場所・住所・店名で検索）
- 中部：マップ表示（ガンプラ取扱店舗をピン表示・現在地表示）
- 右下：現在地ボタン・店舗一覧ボタン

---

## 検索機能

`MKLocalSearchCompleter` によるリアルタイム入力補完に対応する。

- 文字入力中に候補リストを表示（地名・住所・店舗名・POI名すべて対応）
- 候補をタップするとその地点にカメラが移動し、詳細シートを表示する

---

## 店舗登録

検索または地図上のPOIタップから店舗を選択し、ガンプラ取扱店舗として登録する。
手動での緯度・経度入力による登録は廃止し、以下の方法のみ対応する。

| 登録方法 | 操作 |
|----------|------|
| 検索から登録 | 候補タップ後に表示される詳細シートの「ガンプラ巡回店舗に追加」ボタン |
| POIタップから登録 | マップ上のPOI（コンビニ・ショッピングモール等）をタップ後に表示される詳細シートの「ガンプラ巡回店舗に追加」ボタン |

すでに登録済みの店舗の場合は「登録済み」と表示し、重複登録を防ぐ。

---

## POIタップ機能

iOSマップと同様に、マップ上の既存POIアイコンをタップすると詳細シートが表示される。

表示内容：
- 店舗名・住所
- 電話番号（タップで発信）
- Webサイト（タップでブラウザ遷移）
- 「ガンプラ巡回店舗に追加」ボタン

---

## 店舗詳細表示

登録済み店舗のピンをタップするとボトムシート（.sheet モディファイア）が表示され、以下を確認できる。

- 店舗基本情報（名前・座標）
- 再販予定日からの平均ずれ時間
- 品出し時刻一覧（記録がない場合は「記録なし」を表示）
- 品出し時刻「編集」リンク（S13 品出し時刻記録画面へ遷移）
- お気に入り登録・解除ボタン
- 削除ボタン

---

## 現在地機能

- 右下の現在地ボタンタップでマップが現在地に移動する。
- 位置情報の使用許可が必要（初回起動時に許可ダイアログを表示）。
- マップ上に現在地（青いドット）を常時表示する。

---

## お気に入り機能

- お気に入り登録するとマップ上にマークが表示される（赤ピン）。
- 通常店舗はオレンジピンで表示。
- ほしい物リストの購入場所でお気に入り店舗から自動検索できる。

---

## 店舗一覧

「店舗一覧」ボタン押下でお気に入り店舗のリストを表示。
店舗を選択すると S13（品出し時刻記録画面）へ遷移し、以下が可能。

- 品出し時刻の確認・追加・削除
- お気に入り登録・解除はスワイプアクションで操作

---

## 品出し時刻記録

店舗ごとに品出しが行われた時刻を最大5件記録できる。

| 項目 | 仕様 |
|------|------|
| 記録上限 | 1店舗あたり5件まで |
| 入力方法 | ドラムロール式の時刻ピッカー（時・分） |
| 操作 | 右上「編集」ボタンでeditモード切替。editモード中に「時刻を追加」行とゴミ箱アイコンを表示 |
| 削除（スワイプ） | 左スワイプで「削除」ボタン表示。確認なしで即削除 |
| 削除（ゴミ箱） | editモード中のみ各行右端に表示。タップで確認ダイアログ → OKで削除 |
| アクセス経路 | 店舗一覧の行タップ / 登録済み店舗詳細シートの「編集」リンク / 巡回詳細の店舗行タップ |

---

## 品出し差分リセット

店舗ごとに蓄積された品出し差分データを個別にリセットできる。

| 項目 | 仕様 |
|------|------|
| アクセス | S13（品出し時刻記録画面）の「品出し差分」セクション |
| 平均差分表示 | 記録なしの場合は「記録なし」、ある場合は `+X.X時間` 形式 |
| リセット操作 | 「差分データをリセット」ボタン → 確認ダイアログ → OKで該当店舗の全 StockDelayRecord を削除し averageDelayHours を 0 にリセット |
