# F04 ガンプラ巡回

巡回予定の作成・管理・通知を行う機能。

---

## 巡回予定の作成

「巡回予定を作成」ボタン押下で以下の情報を入力する。

| 項目 | 備考 |
|------|------|
| 日にち | 入力欄タップでカレンダーUIを表示して選択 |
| 時刻 | テキスト入力（HH:mm形式） |
| 店舗名 | 店舗分析の登録店舗から選択可 |
| 欲しいガンプラ | 選択した日付の前月・当月・翌月に再販予定日が登録されているアイテムのみ表示。再販予定日未登録のアイテムは非表示。表示形式は「グレード：商品名」 |

保存後、再販カレンダーに巡回予定として登録される。

---

## 巡回予定一覧

- デフォルトは作成順に表示。
- リストをタップして詳細を閲覧可能。

---

## 巡回詳細表示・編集

再販カレンダーの巡回予定日タップまたはリストタップで詳細を表示。

**画面構成**
- 巡回情報（店舗・日付・時間・通知アラーム）
- 対象アイテム一覧
- 削除ボタン

右上の「編集」ボタンタップで巡回予定作成画面（編集モード）をシートで表示し、登録内容を変更可能。

---

## 通知

巡回予定ごとに最大3つの通知タイミングを設定できる。

| 項目 | 仕様 |
|------|------|
| 選択肢 | 5分前・10分前・15分前・30分前・1時間前・2時間前・3時間前 |
| 最大数 | 3つまで複数選択可 |
| デフォルト | 設定なし（通知しない） |
| 実装 | `UNUserNotificationCenter` を使用してスケジュール |
