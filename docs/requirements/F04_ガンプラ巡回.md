# F04 ガンプラ巡回

巡回予定の作成・管理・通知を行う機能。

---

## 巡回予定の作成

「巡回予定を作成」ボタン押下で以下の情報を入力する。

| 項目 | 備考 |
|------|------|
| 日にち | 入力欄タップでカレンダーUIを表示して選択 |
| 時刻 | テキスト入力（HH:mm形式） |
| 店舗名 | 店舗分析の登録店舗から選択可 |
| 欲しいガンプラ | 選択した日付の前月・当月・翌月に再販予定日が登録されているアイテムのみ表示。再販予定日未登録のアイテムは非表示。表示形式は「グレード：商品名」 |

保存後、再販カレンダーに巡回予定として登録される。

---

## 巡回予定一覧

- デフォルトは作成順に表示。
- リストをタップして詳細を閲覧可能。

---

## 巡回詳細表示

再販カレンダーの巡回予定日タップまたはリストタップで詳細を表示。

**画面構成**
- 上部：日にち
- 中部：欲しいガンプラ一覧
- 下部：時間 / 店舗名 / 予定差分（店舗分析の平均ずれ）を時間順に表示

---

## 通知

巡回予定日当日に通知を受け取ることができる（設定でON/OFF可能）。
通知は `UNUserNotificationCenter` を使用してスケジュールする。
